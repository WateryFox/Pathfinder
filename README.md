# mazepathfinding-inPython
A pathfinding algorithm made with Python. Please check README for how to use!

To use you must first click somewhere as a starting point which then an orange box will appear. That is your starting point. 
Secondly do the same thing again as your finishing point which then a cyan box will appear. That is your finish point.
Thirdly you may begin to draw the walls of the maze which will be shown as black box.
Press space to let the program pathfind to its objective!
